tsunami1.txt, tsunami2.txt, and tsunami3.txt are 3 text files that
will be used by the auto grader.
output1.txt, output2.txt, and output3.txt contain the correct output
generated by the program from the 3 input files.
tsunamiall.txt is the concatenation of the 3 input files.
outputall.txt is the correct output generated by the program for
input in tsunamiall.txt.
